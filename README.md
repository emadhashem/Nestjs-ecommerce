## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Database design

1. User Table:
   - User ID
   - First Name
   - Last Name
   - Email
   - Password
   - Address
   - Phone Number

2. Shop Table:
   - Shop ID
   - Shop Name
   - Shop Description
   - Shop Owner ID (Foreign Key)
   - Shop Address
   - Shop Phone Number

3. Product Table:
   - Product ID
   - Product Name
   - Product Description
   - Price
   - Stock Quantity
   - Category
   - Shop ID (Foreign Key)

4. Order Table:
   - Order ID
   - User ID (Foreign Key)
   - Order Date
   - Total Amount
   - Payment Method
   - Delivery Address
   - Delivery Status

5. Order Item Table:
   - Order Item ID
   - Order ID (Foreign Key)
   - Product ID (Foreign Key)
   - Quantity
   - Price

6. Cart Table:
   - Cart ID
   - User ID (Foreign Key)

7. Cart Item Table:
   - Cart Item ID
   - Cart ID (Foreign Key)
   - Product ID (Foreign Key)
   - Quantity

8. Payment Table:
   - Payment ID
   - User ID (Foreign Key)
   - Order ID (Foreign Key)
   - Payment Date
   - Amount
   - Payment Method

9. Review Table:
   - Review ID
   - User ID (Foreign Key)
   - Product ID (Foreign Key)
   - Review Date
   - Rating
   - Comment

10. Notification Table:
   - Notification ID
   - User ID (Foreign Key)
   - Notification Type (e.g. Order Confirmation, Delivery Update, Sale Notification)
   - Notification Message
   - Notification Date
